Farben in CSS - Grundlagen

In CSS können wir Farben auf verschiedene Arten angeben. Frben werden z.B. für Texte, Hintergrundfarben und Rahmen verwendet.

  1. Farbnamen (z.B.: green, red)
  2. HEX-Wert (z.B.: #ff0000)
  3. RGB/RGBA (z.B.: rgb(255, 0, 0))
  4. HSL/HSLA (z.B.: hsl(0, 100%, 50%))

1. Farbnamen

CSS kennt viele eingebaute Farbnamen wie red, green und blue.

Ich bin rot(red).

Ich bin blau(blue).

Ich bin grün(green).

                    .name-red{
                        color: red;         /* Schriftfarbe ist rot */
                    }

                    .name-blue{
                        color: blue;        /* Schriftfarbe ist blau */
                    }

                    .name-green{
                        color: green;       /* Schriftfarbe ist grün */
                    }

Einfach, abrr begrenzte Auswahl und oft nicht exakt der gewünschte Farbton.

2. HEX-Farben (#RRGGBB)

HEX-Farben sind weit verbreitet im Web. Sie bestehen aus #RRGGBB (Rot, Grün, Blau). Die Werte im Hexadezimal gehen von 00 bis FF, dabei steht 00 = 0 (nichts) und FF = 255 (volle Stärke).

Ich bin #ff0000 (ROT).

Ich bin #00aa00 (GRÜN).

Ich bin #0000ff (BLAU).

                    .hex-red{
                        color: #ff0000;     /* Schriftfarbe ist rot */
                    }

                    .hex-green{
                        color: #00aa00;     /* Schriftfarbe ist grün bei #00ff00 = neongrün*/
                    }

                    .hex-blue{
                        color: #0000ff;     /* Schriftfarbe ist blau */
                    }

HEX-Farben sind sehr genau und Standard Design in der Webentwicklung.

3. RGB/RGBA

RGB mischt Rot, Grün ud Blau (0-255).
RGBA hat zusätzlich einen Alpha-Wert (Transparent von 0 bis 1).

Ich bin rgb(255, 0, 0).

Ich bin rgba(255, 0, 0, 0.4)

Ich bin rgb(0, 255, 0)

Ich bin rgba(0, 255, 0, 0.5)

Ich bin rgb(0, 0, 255)

Ich bin rgba(0, 0, 255, 0.6)

                    .rgb-red{
                        color: rgb(255, 0, 0);
                    }

                    .rgba-red{
                        color: rgba(255, 0, 0, 0.4);
                    }

                    .rgb-green{
                        color: rgb(0, 255, 0);
                    }

                    .rgba-green{
                        color: rgba(0, 255, 0, 0.5);
                    }

                    .rgb-blue{
                        color: rgb(0, 0, 255);
                    }

                    .rgba-blue{
                        color: rgba(0, 0, 255, 0.6);
                    }

Super für Transparenz (Overlayers/ Effekte)

4. HSL/HSLA

HSL steht für: Farbton(Hue), Sättigung(Saturation), Helligkeit(Lightness).
hsla (Farbton, Sättigung, Helligkeit, Alpha-Wert)

  1. Welche Farbe?
  2. Wie kräftig?
  3. Wie hell?

Ich bin hsl(0, 100%, 50%)

Ich bin hsla(0, 100%, 50%, 0.7)

Ich bin hsl(120, 100%, 40%)

Ich bin hsla(120, 100%, 40%, 0.5)

Ich bin hsl(240, 100%, 30%)

Ich bin hsla(240, 100%, 30%, 0.3)

                    .hsl-red{
                        color: hsl(0, 100%, 50%);
                    }

                    .hsla-red{
                        color: hsla(0, 100%, 50%, 0.7);
                    }

                    .hsl-green{
                        color: hsl(120, 100%, 40%);
                    }

                    .hsla-green{
                        color: hsla(120, 100%, 40%, 0.5);
                    }

                    .hsl-blue{
                        color: hsl(240, 100%, 30%);
                    }

                    .hsla-blue{
                        color: hsla(240, 100%, 30%, 0.3);
                    }

Sehr intuitiv für Design (hellere/dunklere stellen oder Farbton können wir drehen).
