CSS Farbverläufe (Gradients)

Mit CSS kannst du Farbverläufe als Hintergrund verwenden. Farbverläufe sind keine Bilder, sondern werden direkt von CSS erzeugt.

Die wichtigsten Farbverläufe:

  1. Linear Gradient = Verlauf in eine Richtung
  2. Radial Gradient = Verlauf von der Mitte nach außen
  3. Conic Grandient = kreisförmiger Verlauf

1. Linear Gradient linear-gradient

Ein linearer Verlauf, geht in eine Richtung (z.B.: von oben nach unten).

linear-gardient (oben nach unten) 
                    .linear-basic{
                        background: linear-gradient(#00a8ff, crimson);      -> #00a8ff = erste Farbe (oben) | crimson = zweite Farbe (unten)
                    }

Du kannst auch eine Richtung angeben:

linear-gardient (nach links) 
                    .linear-direction{
                        background: linear-gradient(to left, #6465a5, #f05837 );        -> nach "to" kommt die Richtung in die es Verlaufen soll
                    }

Wir können auch einen Winkel benutzen:

linear-gardient (350deg) 
                    .linear-angle{
                        background: linear-gradient(350deg, #00743f, #f2a104);      -> zahl+deg = gibt den Winkel an, der genutzt werden soll
                    }

2. Radial Gradiant (radial-gradient)

Der Farbverlauf geht von der Mitte nach außen (kreisförmig)

radial-gradient 
                    .radial-basic{
                        background: radial-gradient(circle, #c0334d, #f1931b);      -> circle weil es kreisförmig sein soll
                    }

Du kannst die Form beeinflussen:

radial-gradient(ellipse) 
                    .radial-ellipse{
                        background: radial-gradient(ellipse, #212027, #f22f08);     -> ellipse weil es ellipsenförmig ist
                    }

3. Conic Gradient (conic-gradient)

Ein conic-gradient ist ein kreisförmiger Verlauf um einen Mittelpunk. Er eignet sich gut für z.B.: Diagramme und Deko.

conic-gradient 
                    .conic-basic{
                        background: conic-gradient(#003d73, #0878a4, #1ecfd6, #edd170, #c05640 );
                    }
