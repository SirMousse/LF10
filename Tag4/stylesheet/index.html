CSS einbinden - Inline, Intern und Extren

Es gibt drei Arten, CSS in HTML einzubinden:

  1. Inline CSS = direkt im HTML Element
  2. Interne CSS = im <style>- Tag im Head drin
  3. Externe CSS = in einer .css Datei angelegt

1. Inline CSS

Inline CSS wird direkt im HTML Tag mit dem Attribut style geschrieben.

Diese Text ist mit Inline CSS gestylt. 

                    <p style="color: red;                        -> color = Farbe der Schrift        
                                background-color: antiquewhite;     -> background-color = ist die Hintergrundfarbe
                                padding: 8px;"                      -> padding = ist der Innenabstand (Platz zwischen dem Inhalt eines Elements und seinem Rand)
                            >
                    Diese Text ist mit Inline CSS gestylt.
                    </p>

Wann ist das sinnvoll? Fast nie. Nur für schnelle Tests oder einzelne Ausnahmen.
Warum ist es schlecht? Unübersichtlich und sehr schlecht wartbar.

2. Interne CSS (im <style>- Tag)

Internes CSS steht im <style>- Tag der HTML Datei. Es gilt für die gesamte Seite.

Dieser Bereich ist mit internem CSS gestylt.

                    <style>
        Erklärung:      Selektor{
                            eigenschaft: wert;
                        }

                        body{
                            font-family: Arial, Helvetica, sans-serif;      -> bestimmt die Schriftart
                            line-height: 1.5;                               -> bestimmen den Zeilenabstand 
                        }

                        .intern{
                            color: darkgreen;                               -> color = Farbe der Schrift
                            background-color: #e6ffe6;                      -> background-color = ist die Hintergrundfarbe
                            padding: 8px;                                   -> Innenabstand
                        }

                        .box{
                            border: 2px black solid;                        -> border ist der Rahmen = 2px ist die stärke des Rahmens / black ist die Farbe des Rahmens / solid bedeutet ist ein fester Rahmen
                            padding: 10px;                                  -> padding = Innenabstand
                            margin: 10px 0;                                 -> margin = Außenabstand (oben, rechts, unten, links) | 10 px steht für oben + unten & 0 steht für 0pc und rechts und links
                        }
                    </style>

Wann ist das sinnvoll? Bei kleineren Projekten und Übungen.
Warum? Alles in einer Datei ist und leicht zu verstehen.

3. Externe CSS (seperate Datei)

Externe CSS liegt in einer eigenen Datei (z.B.: styles.css) und wird mit dem <link> eingebunden.

Dieser Bereich ist mit externem CSS gestylt.

                    .extern{
                        color: antiquewhite;            -> color = Schriftfarbe
                        background-color: #3333cc;      -> background-color = Hintergrundfarbe
                        padding: 10px;                  -> padding = Innenabstand
                    }

Wann ist das sinnvoll? Fast immer bei "echten" Webseiten.
Warum? Sauber, wiederverwendbar und leicht wartbar.

4. Priorität und Reihenfolge

Wenn mehrere Styles auf das selbe Element wirken, gilt:

  1. Inline CSS (höchste Priorität)
  2. Interne CSS
  3. Extrene CSS (niedrigste Priorität)
